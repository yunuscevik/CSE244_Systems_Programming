Usage: ./withPipeandFIFO 'string'  <directoryName>

Terminalden make ile olusturdugumuz dosyayi calistirirken usage a uygun sekilde 
calistirmamiz gereklidir.
Example:  ./withPipeandFIFO ece dirA   gibi yazildigi taktirde calismaktadir.
make clean yapildigi taktirde "rm *.o listdir *.log" calisip bu dosyalar silinecektir.
